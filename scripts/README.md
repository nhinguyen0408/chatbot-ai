# THƯ MỤC KỊCH BẢN TƯ VẤN

Thư mục này chứa các file kịch bản tư vấn cho chatbot AI.

## Các file kịch bản

### 1. `kich-ban-vbi.txt`
File kịch bản đầy đủ với thông tin chi tiết về:
- Bảo hiểm TNDS xe cơ giới
- Bảo hiểm VBICare
- Bảo hiểm du lịch

Bao gồm: câu hỏi thường gặp, quyền lợi, mức phí, quy trình tư vấn, và các tình huống xử lý.

### 2. `kich-ban-vbi-sample-short.txt`
File kịch bản rút gọn để test nhanh, chỉ chứa thông tin cơ bản về các sản phẩm.

## Format kịch bản

Kịch bản được viết theo format Markdown với cấu trúc:
- Headers (#, ##, ###) để phân chia sản phẩm và mục
- Q&A format cho câu hỏi thường gặp
- Bullet points cho danh sách quyền lợi
- Code blocks cho các ví dụ cụ thể

## Cách sử dụng

1. Upload file kịch bản vào thư mục này
2. Hệ thống sẽ tự động parse và load vào Gemini
3. Chatbot sẽ sử dụng nội dung kịch bản để trả lời khách hàng

## Lưu ý

- File kịch bản nên được viết bằng tiếng Việt có dấu
- Format rõ ràng, dễ đọc để AI có thể parse chính xác
- Cập nhật thông tin định kỳ để đảm bảo tính chính xác
